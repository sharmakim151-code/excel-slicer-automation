# excel-slicer-automation
Interactive Excel dashboard using VBA to automate slicer control across multiple PivotTables, with SQL and Power BI integration for sales analysis.
# 📊 CLD Sales Analytics Dashboard

## 🚀 Overview
This project is an interactive sales analytics dashboard built using Excel, SQL, Power BI, and basic Python. It focuses on analyzing counselor performance, regional trends, and revenue insights.

The key highlight of this project is the use of VBA to dynamically control slicer connections across multiple PivotTables using checkboxes, reducing manual effort and improving usability.

---

## 💡 Problem Statement
Managing multiple PivotTables with slicers in Excel often requires manual filtering and repetitive effort.

### ✅ Solution
Developed a VBA-based approach to dynamically connect/disconnect PivotTables from slicers using checkboxes, enabling faster and more flexible data interaction.

---

## 🔹 Key Features
- Interactive Excel dashboard with Pivot Tables & Charts  
- Checkbox-based control for multiple PivotTables  
- Dynamic slicer automation using VBA  
- Region-wise filtering and performance tracking  
- Power BI dashboard for advanced visualization  
- Data cleaning and preprocessing using Python (Pandas - basic)
